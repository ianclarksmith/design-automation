LinetypeCount


Returns the number of linetypes in the document.


Syntax


Rhino.LinetypeCount ()


Parameters


None.


Returns



Number 

The number of linetypes in the document.


Example


Dim intCount

intCount = Rhino.LinetypeCount

Rhino.Print "There are " & CStr(intCount) & " linetypes."
