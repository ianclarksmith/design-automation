Toolbar Methods


The following methods are available for querying or manipulating toolbar collections.




Method 

Description

AddToolbar 

Adds a new toolbar to a toolbar collection.

AddToolbarButton 

Adds a new button to a toolbar.

AddToolbarCollection 

Adds a new toolbar collection.

CloseToolbarCollection 

Closes a open toolbar collection.

DeleteToolbar 

Deletes a toolbar from a toolbar collection.

HideToolbar 

Hides a visible toolbar.

IsToolbar 

Verifies the existence of a toolbar in a toolbar collection.

IsToolbarCollection 

Verifies the existence of a toolbar collection.

IsToolbarDocked 

Verifies a visible toolbar is docked.

IsToolbarVisible 

Verifies a toolbar is visible.

OpenToolbarCollection 

Opens a toolbar collection.

SaveToolbarCollection 

Saves a toolbar collection.

SaveToolbarCollectionAs 

Saves a toolbar collection as a different name.

ShowToolbar 

Displays a toolbar.

ToolbarCollectionCount 

Returns the number of toolbar collections.

ToolbarCollectionNames 

Returns the names of all toolbar collections.

ToolbarCollectionPath 

Returns the path to a toolbar collection.

ToolbarCount 

Returns the number of toolbars in a toolbar collection.

ToolbarNames 

Returns the names of all toolbars in a toolbar collection.
