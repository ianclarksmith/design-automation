[AliasCount]

    Returns the number of command alias in Rhino.

[Syntax]

    Rhino.AliasCount ()

[Parameters]

    None.

[Returns]

    Number
    True or False indicating success or failure.

[Example]

    MsgBox Rhino.AliasCount
