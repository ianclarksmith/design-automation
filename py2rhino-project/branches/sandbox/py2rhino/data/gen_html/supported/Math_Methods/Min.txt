Min



Min


Returns the minimum number from an array of numbers.


Syntax


Rhino.Min (arrNumbers)


Parameters



arrNumbers 

Required.  Array.  An array of numbers to analyze.


Returns



Number 

The minimum number if successful.

Null 

If not successful, or on error.


Example


MsgBox Rhino.Min(Array(1,2,3,4,5,6,7,8,9))
