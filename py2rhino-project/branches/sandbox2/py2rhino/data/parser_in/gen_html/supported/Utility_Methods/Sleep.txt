Sleep



Sleep


Suspends the execution of a running script for the specified interval.


Syntax


Rhino.Sleep (lngMilliseconds)


Parameters



lngMilliseconds 

Required.  Number.  The duration in milliseconds.


Returns



Null 

If successful, or on error.


Example


Rhino.Print "This"

Rhino.Sleep 2000

Rhino.Print "is"

Rhino.Sleep 2000

Rhino.Print "a"

Rhino.Sleep 2000

Rhino.Print "slow"

Rhino.Sleep 2000

Rhino.Print "message!"
