explode(self, objects, delete=pythoncom.Empty)

source code 

Explodes, or un-joins, one more polysurface objects. Polysurfaces will be exploded into separate surfaces.

Parameters

objects (list of array_of _ObjectRoot, Required) - A list of polysurface objects to explode. delete (boolean, Optional) - Delete input objects after exploding. The default is not to delete objects (False).

Returns

list of objects - A list of strings identifying the newly created surface objects if successful. None - If not successful, or on error.

Rhinoscript

This function calls the Rhinoscript function: ExplodePolysurfaces
