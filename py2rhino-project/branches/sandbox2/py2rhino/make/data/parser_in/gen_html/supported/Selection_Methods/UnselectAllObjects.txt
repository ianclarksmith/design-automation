UnselectAllObjects


Unselects all objects in the document.


Syntax


Rhino.UnselectAllObjects


Parameters


None.


Returns



Number 

The number of objects that were unselected.


Example


Dim intObjects

intObjects = Rhino.UnselectAllObjects

Rhino.Print CStr(intObjects) & " objects were unselected"
