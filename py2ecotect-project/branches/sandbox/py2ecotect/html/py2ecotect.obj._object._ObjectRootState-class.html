update(self)

source code 

Use this command to check and refresh inter-object relationships for the specified object. Complex model with many inter-relationships can take a little time to regenerate so ECOTECT doesn't do this automatically after each script-based manipulation. You can also use model.update to do this for all objects in the model at once.

Parameter(s) There are no parameters for this command.
