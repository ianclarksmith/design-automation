load(filename='')

source code 

Loads a weather data file into the current model. This script method only loads the actual hourly weather data, leaving the latitude, longitude, time-zone and other location data untouched.

Parameter(s) This command takes the following parameters.

[filename] This optional parameter indicates the full path to the file that is to be loaded. If not given, then the last loaded weather file will be used.

load_all(filename='')

source code 

Loads a weather data file into the current model. This script method loads the actual hourly weather data, and the latitude, longitude, time-zone and other location data.

Parameter(s) This command takes the following parameters.

[filename] This optional parameter indicates the full path to the file that is to be loaded. If not given, then the last loaded weather file will be used.

source code 

Retrieves the beam solar radiation (W) for the specified day and hour in the current weather data.

Parameter(s) This property takes the following parameters.

day An integer value between 1 and 365 representing the day of the year.

hour An integer value between 0 and 23 representing the hour of the day.

Return Value(s) Getting this property returns the following value(s).

result A value in Watts representing the beam solar radiation.
