source code

Parameter(s) This property takes the following parameters.

latitude The latitude in decimal degrees.

longitude The longitude in decimal degrees.

timezone The timezone as a GMT offset in decimal hours.

[update] If this optional parameter is set, the main window will be updated with a redrawn view and sunpath. This is a boolean value where 1 or true represents the affirmative and 0 or false the negative.
